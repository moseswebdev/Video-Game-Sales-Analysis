# 🎮 Video Game Sales Analysis

## 🚀 Overview
This project analyzes global video game sales data to identify trends across platforms, genres, and publishers.

## 🎯 Objectives
- Identify top-selling games and platforms
- Analyze sales trends over time
- Explore regional sales distribution

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib, Seaborn

## 📈 Key Insights
- Certain platforms dominate specific time periods
- Action and Sports genres lead in sales
- Regional preferences differ significantly
